Task 7 - Sales Summary Project

Files Included

1.  sales_summary.pdf
    -   Contains SQL query, output table, and bar charts.
2.  task7_sales_summary.py
    -   Python script that connects to SQLite, runs queries, prints
        summary, and generates charts.

How to Run the Script

1.  Place sales_data.db in the same folder as the .py file.

2.  Install required libraries (if needed):

        pip install pandas matplotlib

3.  Run:

        python task7_sales_summary.py

Output

-   Printed summary on console
-   Bar chart displayed using matplotlib

Purpose

This task demonstrates: - Basic SQLite usage - SQL queries inside
Python - Using pandas for data loading - Creating basic bar charts
